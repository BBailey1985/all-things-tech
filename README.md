# All Things Tech

## Description
This app is a blog all about tech/coding things.

  ## Link to deployed app
  https://ancient-hamlet-74658.herokuapp.com/
  
  ## Installation
  To install this app you will need 
  * Express
  * MySQL2
  * Dotenv
  * Sequelize
  * Handlebars
  * Express-Handlebars
  * Bcrypt
  * Connect-Session-Sequelize
  * Express-Session
  
  ## Usage
  To keep track of all things tech!
